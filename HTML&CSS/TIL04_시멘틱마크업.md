### 웹 서비스 만드는 과정

0. (기획자) 서비스 기획 -> 1. (백엔드) URL 구조 -> 2. (백엔드) model 설계 : 게시판..게시물 제목, 내용, 사진 등.... -> 3. (디자이너) 디자인 -> 4. (프론트엔드) templates 작성(HTML)
